All about git

It's as easy as One Two Three!

Git Add

$git add filename

Tell Git to watch whatever file name is specified. You can add all files at the same time using the command $ git add .

Git Commit

$git commit -m "Your commit message here"

Make your changes official and create a new commit. This will allow you to then push your changes to the cloud. You can think of it as a fallback point as well because you can always return to the state this code was in as it was at this commit.

Git push

$git push origin master

Sends the current commit up to the cloud.
